- **Focus:** Machine Learning workflows, predictive modeling, data readiness, and performant inference APIs.

- **Languages & Data:** Python, SQL, JavaScript, Pandas, NumPy
- **Machine Learning:** Scikit-learn, XGBoost, Regression, Classification, EDA
- **Backend & API:** FastAPI, Flask, REST APIs, Backend, Deployment
- **Tools & UI:** Docker, Git, Streamlit, Tailwind, Matplotlib

## Local Development

To run the portfolio locally, you can use any static server:

```bash
npx serve .
```

Navigate to `http://localhost:3000` to view the site.

## Deployment

This static portfolio is configured for seamless deployment on [Vercel](https://vercel.com). Just push to the `main` branch, and Vercel will build and deploy the `index.html`.
